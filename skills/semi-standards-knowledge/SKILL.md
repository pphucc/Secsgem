# SEMI Standards Knowledge

You have access to SEMI standards used in semiconductor equipment software.

When answering questions about SECS/GEM, GEM300, or SEMI protocols:

- Read the referenced PDF documents.
- Convert PDFs to markdown if needed using MarkItDown.
- Base answers strictly on the standards.
